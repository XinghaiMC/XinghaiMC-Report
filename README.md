# XinghaiMC-Report
## 简介
一款简单清晰明了的举报插件。
## 指令
/report 玩家 原因
- 举报玩家
/handle
- 查看举报列表
/handle ID
- 举报处理完成
## 权限
xinghaimc.report 用于举报的权限
xinghaimc.report.handle 用于处理举报的权限
